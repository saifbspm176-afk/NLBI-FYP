About This Project

Final Year Project: Natural Language for Business Intelligence

This application provides comprehensive access to IIU's institutional documents including the complete Policy Compendium (2020-2023) and all IIU Statutes (2006, 2005, 2004, 2000, 1992, 1987). Built with modern web technologies for fast, intuitive exploration of university policies and regulations.

Core Features

  • ✓ Real-time full-text search
  • ✓ Hierarchical document browsing
  • ✓ Auto-generated Q&A pairs
  • ✓ Complete IIU Statutes (all versions)
  • ✓ Detailed policy compendium
  • ✓ JSON data export
  • ✓ Mobile responsive design
